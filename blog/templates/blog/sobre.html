Sobre o DevBlog

O DevBlog é um projeto desenvolvido pela Turma Python como o trabalho final e conclusão do curso.

Este espaço funciona como um blog, vlog ou diário em texto e vídeo, onde desenvolvedores documentam todo o processo de criação de jogos, aplicativos ou softwares.

A nossa trajetória na Turma Python no Senac Tech de Porto Alegre foi dividida em 5 módulos fundamentais:

UC1 - Lógica de Programação

Definir algoritmos estruturados em lógica de programação. Nosso primeiro contato, onde aprendemos o conceito básico sobre lógica utilizando Portugol.

UC2 - Algoritmos em Python

Desenvolver algoritmos em Python. Revisitamos a lógica de programação básica, mas desta vez, aplicando diretamente na linguagem Python.

UC3 - Aplicações Desktop

Desenvolver aplicações desktop em Python. Hora de sair dos fundos da casa (backend) e começar a integrar o frontend em nosso código de programação.

UC4 - Aplicações Web Responsivas

Desenvolver aplicações web responsivas com Python. Preparação para a reta final da jornada, aprofundando os estudos em aplicações web.

UC5 - Websites com Django

Planejar e desenvolver websites utilizando o framework Django. O último passo da nossa aventura, onde criamos este projeto em conjunto: o DevBlog!

🚀 Esta foi a nossa jornada como alunos e membros da Turma de Python no Senac Tech!
